Active / Death Trigger

afterOwnAttacks: 3 คือสะสมการโจมตีปกติ 3 ครั้งแล้วเอฟเฟกต์ทำงาน, onBattleStart: true คือเริ่มไฟต์, และ onDeath: true หรือ triggerOn: "death" คือทำงานเมื่อตัวละครเจ้าของสกิลตาย รองรับการเลือกเป้าหมายครบชุดเหมือนเอฟเฟกต์อื่น

คัดลอกและขโมยสกิล

ต้านสถานะ / ต้านผิดปกติ

abnormalStatusResistance ไม่ต้องใส่ value และต้านสถานะผิดปกติแบบสมบูรณ์ทันที ระบุเฉพาะสถานะได้ด้วย statuses หรือ resistStatuses เช่น stun, freeze, death, skill-cancel; ไม่ระบุ = ต้านทั้งหมด ส่วน statusResistance เดิมยังใช้เปอร์เซ็นต์พอยต์ได้เหมือนเดิม
